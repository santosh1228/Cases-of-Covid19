# Build a Corona Meter App which displays the Number of Active corona cases in a Particular Country

## Tools Used - HTML, CSS, Javascript

## This App will Give you Information about
- Country Name
- Country Code
- New Covid Cases
- New Covid Deaths
- Total Covid Cases
- Total Covid Deaths
